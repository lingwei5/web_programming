HTML, CSS & JavaScript

构建现代网页的三大核心技术

HTML - 骨架

定义网页的结构和内容，如标题、段落、图像等

CSS - 外观

控制网页的样式和布局，如颜色、字体、间距等

JavaScript - 行为

实现网页的交互和动态功能，如表单验证、动画等

互动演示

在下方查看代码并体验三者的协同工作方式

网页代码示例
<!-- HTML 代码 -->
<div id="card">
  <h2>欢迎来到前端世界</h2>
  <p>这是一个演示HTML、CSS和JavaScript如何协同工作的示例。</p>
  <button class="btn">点我!</button>
</div>

/* CSS 代码 */
#card {
  width: 300px;
  padding: 20px;
  border-radius: 10px;
  background: white;
  box-shadow: 0 5px 15px rgba(0,0,0,0.1);
  transition: all 0.3s ease;
}

.btn {
  padding: 10px 20px;
  background: #3498db;
  color: white;
  border: none;
  border-radius: 5px;
  cursor: pointer;
}

// JavaScript 代码
const card = document.getElementById('card');
const btn = document.querySelector('.btn');

btn.addEventListener('click', () => {
  // 改变卡片样式
  card.style.transform = 'scale(1.05)';
  card.style.backgroundColor = '#f0f0f0';
  
  // 添加新元素
  const message = document.createElement('p');
  message.textContent = '你点击了按钮!';
  card.appendChild(message);
});
